# webscrap-Disney-Bot
Telegram Webscrap Bot 

## Configs

* TOKEN  - Get bot token from [BotFather](https://t.me/botfather)

* API_ID     - From [my.telegram.org](https://my.telegram.org)

* API_HASH    - From [my.telegram.org](https://my.telegram.org)
